---
title: Show Amazon Linux AMI version using a command
tags: aws
---

`/etc/issue`
---

```bash
$ cat /etc/issue
Amazon Linux AMI release 2015.09
Kernel \r on an \m
```

`/etc/system-release`
---

```bash
$ cat /etc/system-release
Amazon Linux AMI release 2015.09
```
